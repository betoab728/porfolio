Sistema de Control de Asistencia con Reconocimiento Facial IA

Este sistema tiene como propósito automatizar y optimizar el proceso de registro de asistencia en entornos laborales o educativos, reduciendo los errores humanos y mejorando la fiabilidad en la toma de registros. El objetivo principal es crear una plataforma en la que se pueda gestionar la asistencia de manera precisa y en tiempo real mediante el reconocimiento facial.

Tecnologías Utilizadas

Características Principales

  • Gestión de usuarios con autenticación segura
  • Registro de trabajadores
  • Registro de asistencia mediante reconocimiento facial
  • Reporte dinámico de asistencias en formato PDF

Retos y Soluciones

El desarrollo del Sistema de Control de Asistencia con Reconocimiento Facial presentó desafíos como optimizar el procesamiento de imágenes en tiempo real utilizando Python y OpenCV, diseñar una base de datos eficiente en MySQL para gestionar asistencias y ajustar los algoritmos de IA para diversas condiciones de iluminación y expresiones faciales. Además, se trabajó en crear una interfaz intuitiva y garantizar una integración fluida con dispositivos de captura, logrando un sistema preciso, eficiente y fácil de implementar.
